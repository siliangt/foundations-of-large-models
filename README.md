# 《大模型基础》课程教材与资料

本仓库用于维护《大模型基础》（Foundations of Large Models）课程的教材初稿、配套实验、参考资料与课程建设文档。

## 目录结构

- `textbook/`：教材章节的可编辑 Word 文档
- `labs/`：实验讲义、代码与说明
- `resources/`：课程申请、阅读材料与参考资源索引
- `outputs/`：本地生成的临时交付物（不纳入版本控制）

## 教材目录

1. 大模型导论：范式、能力与边界
2. 语言、文本与 Tokenization
3. 神经网络与词表示基础
4. 序列建模与语言模型
5. 注意力机制
6. Transformer 基础架构
7. Transformer 架构演化
8. 预训练：数据、目标与 Scaling Law
9. 大模型训练系统与资源核算
10. 参数高效微调与领域适配
11. 模型压缩、推理与部署
12. 强化学习基础
13. 大模型后训练与对齐
14. 大模型评测与可信性
15. 推理模型、检索增强与多模态大模型
16. 智能体与大模型应用工程

## 维护约定

每完成一个章节或课程资料的实质更新，均应附带清晰的提交说明，并同步至远程 GitHub 仓库。
